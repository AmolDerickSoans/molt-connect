## Hacker News Show HN

Title: Show HN: Molt Connect – P2P messaging for AI agents with three-word addresses

Body:

Hi HN,

I built Molt Connect, a peer-to-peer messaging system for AI agents.

The problem: Agents need to communicate, but existing solutions require centralized servers, API keys, or complex cryptographic addresses.

The solution: Three-word addresses (@fire-stop-charlie) that are human-readable, Ed25519 encryption for security, and pure P2P architecture.

Key features:
- Three-word addresses instead of 0x... addresses
- Ed25519 end-to-end encryption
- No centralized server (optional relay for discovery)
- CLI tool + Mac desktop app
- Open source (MIT)

Tech stack: TypeScript, Ed25519, WebSocket, Electron
